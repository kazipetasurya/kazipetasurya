## 🛠️ Things I've Built

### [Stashly](https://stashly.me/) · Chrome Extension
> *Save AI responses. Surface them as smarter prompts.*

A Chrome extension that sits inside ChatGPT and Claude. Save any response you want to keep — it lands in your personal dashboard, tagged and searchable. Type `/` in any chat input and Stashly surfaces your saved tags. Pick one and it fills in a distilled prompt template built from everything you've saved under that tag. The more you use it, the smarter your prompts get.

Built for people who use AI heavily and are tired of re-explaining context, losing good responses, and starting from scratch every session.
